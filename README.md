# NatureSip - Premium Hostable Application

A premium, high-performance web application for **NatureSip (Hari-Kripa)**, built with React, Vite, and Framer Motion. This application showcases NatureSip's organic fruit and vegetable powders made using advanced flash-drying technology.

## 🚀 Quick Start

### 1. Environment Note
I have installed Node.js (LTS) on this machine using `winget`. If `npm` is not recognized in your terminal, please **restart your terminal** or add `C:\Program Files\nodejs` to your PATH.

### 2. Run Locally (Development)
```powershell
# In a new terminal:
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## 🏭 Manufacturing & Production

NatureSip uses state-of-the-art **Flash Drying Technology** to preserve nutrients.
- **Processing Time:** 15-30 seconds.
- **Temperature:** 40°C - 55°C (Low temperature to preserve enzymes).
- **Sweetening:** Only natural Dry Honey Powder used.
- **Purity:** 100% natural, no preservatives, no artificial sugar.

Refer to `docs/manufacturing_process.md` for full measurements and planning details.

## ✨ Premium Features

- **Rich Aesthetics:** HSL-based color palette (Forest Green & Gold).
- **Glassmorphism:** Modern UI elements with blur effects.
- **Micro-animations:** Smooth transitions using Framer Motion.
- **Direct Media Integration:** High-quality videos and images stored locally for zero-latency loading.
